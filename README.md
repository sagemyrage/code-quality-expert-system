# Code Quality Expert System

Проект по теме: **«Экспертная система оценки качества программного кода с использованием нечёткой логики»**.

## Описание
Web-приложение на Go для оценки качества программного кода на языке Go.  
Система собирает метрики исходного кода и анализирует их с помощью нечёткой логики.

## Стек
- Go
- PostgreSQL
- Redis
- Docker

## Цель проекта
Разработать web-приложение, которое позволяет:
- регистрировать пользователей;
- выполнять проверку кода;
- сохранять результаты проверок;
- вычислять итоговую оценку качества кода на основе метрик и нечёткой логики.
